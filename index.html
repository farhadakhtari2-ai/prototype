SOP-01 — Starting a new project in Autodesk Construction Cloud (ACC)

Workflow Toolkit · Standard Operating Procedure · Published · Version 1.2 · Approved by S. O'Brien, BIM Coordinator

PublishedACC / CDE setupAll disciplines
This SOP must be completed before any project files are uploaded. ACC is the IDS Common Data Environment — the local server is no longer the primary storage location.
Purpose & scope

This procedure describes how to set up a new project within Autodesk Construction Cloud (ACC) as the IDS Common Data Environment, in line with ISO 19650-2 requirements. It covers project creation, folder structure, permission management and initial container configuration.

Step-by-step procedure
1
Log in to Autodesk Construction Cloud
Navigate to acc.autodesk.com and sign in with your IDS Autodesk account. Contact IT Support if you need an account created.
Always use your IDS corporate email — do not use personal Autodesk accounts.
2
Create a new project hub
From ACC home, click New Project. Enter the project name using the IDS code format and set location to Ireland.
Example: IDS-2401 Cork City Library Extension
3
Configure the ISO 19650 folder structure
Create these top-level CDE zones in the Files module:
01_WIPWork in progress (S0)
02_SHAREDCoordination (S1)
03_PUBLISHEDInformation / Review / Construction (S2–S4)
04_ARCHIVESuperseded (S6)
4
Create discipline sub-folders
Within each zone, create sub-folders for each discipline role code (A / S / M / E / C / X). Repeat this structure for all four zones.
5
Add team members and set permissions
Invite all team members via Members settings. BIM Coordinator = Admin. All team members = Editor (own WIP only) + Viewer (other disciplines' Shared/Published).
6
Upload the BIM Execution Plan (BEP) first
The BEP must be the first container uploaded to 03_PUBLISHED / X with suitability S2.
IDS-2401_IDS_ZZ_ZZ_RS_X_0001
7
Confirm setup with BIM Coordinator
Notify S. O'Brien for a quick review. After sign-off, the project is live and all team members can begin uploading WIP containers.
Setup complete — the project is now ready for ISO 19650-compliant file management in ACC.

FAQ & Support
Quick answers for common questions — searchable knowledge base for all IDS staff.
Common questions
Where do I save project files?
All project files must be saved to Autodesk Construction Cloud (ACC) — not the local server. Navigate to the IDS project hub, find your project, and upload using the correct ISO 19650 naming format. See SOP-01 in the Workflow Toolkit for full setup instructions.
How do I name a Revit model correctly?
Use the ISO 19650-2 format: IDS-2401_IDS_ZZ_ZZ_MO_A_0001.rvt. The type code for a Revit model is always MO. Use the Upload Document button in Project Files to auto-generate the correct name.
What is the latest drawing template?
The current IDS title block template is v3.2. Download it from the Discussions section. Always check the hub for the current version before starting a new drawing set.
How do I issue a revised drawing?
Increment the revision in filename and title block (P01 → P02…). Upload to ACC WIP with S0. Change to S1 for review, then S2 to issue. Use ACC transmittal to notify recipients.
Who do I ask for BIM support?
Contact BIM Coordinator S. O'Brien as your first point of contact, or post a question in the Discussions section. For urgent ACC access issues, contact IT Support.
